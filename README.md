# Why use Scythean?
- <p>Scythean has native Linux and MacOS support, is open-source, completely free, and takes up minimal storage space with the Pi version. This means that almost anybody can use Scythean, and it has no chance of being malware because the code is entirely open source and can be viewed and analysed.</p>
# What does it do?
- <p>Scythean allows users to change their vanilla Roblox installation in many ways. Ever wanted to use the old Roblox mouse? Well, you can! Ever wanted to change the moon to be Barack Obama? Well, probably not, but you can!</p>
# Why not just change the files manually?
- <p>You can, but Roblox won't let you keep it there. Every single update, every change you make to the files is completly wiped and reset by Roblox. However with Scythean, it detects for any changes when you open Scythean and instantly resets the files back to your preferences.</p>
# Does it work for Windows?
- <p>Yes, but some features might come late as Scythean is mainly meant to be a MacOS and Linux program.</p>
